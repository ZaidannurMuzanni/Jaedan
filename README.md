## Deskripsi
Praktikum ini bertujuan untuk memahami dasar pengujian API menggunakan Postman dengan metode HTTP:
- GET
- POST
- PUT
- DELETE

API yang digunakan adalah JSONPlaceholder.

Base URL:
https://jsonplaceholder.typicode.com

## Hasil Praktikum

### 1. GET – Get All Posts
Berhasil mengambil seluruh data postingan.
Status: 200 OK

### 2. GET – Get Post by ID
Mengambil data postingan berdasarkan ID.

### 3. POST – Create Post
Menambahkan data postingan baru (simulasi).

### 4. PUT – Update Post
Mengubah data postingan.

### 5. DELETE – Delete Post
Menghapus data postingan.

## Kesimpulan
Pengujian API dengan Postman membantu memahami cara kerja komunikasi data antara client dan server menggunakan format JSON.
